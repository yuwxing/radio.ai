Radio.AI 智能语音电台 - 安装指南
=================================

此文件夹包含完整的Radio.AI应用，具有用户登录功能和语音/文字控制的智能电台系统。

文件列表：
1. login.html - 登录页面
2. index.html - 主应用页面（带用户认证）
3. 本说明文档

安装步骤：
1. 将这三个文件复制到 D:\AI收音机 文件夹中
2. 双击打开 login.html 即可开始使用
3. 使用以下任一方式登录：
   - 演示账号: 用户名 "demo"，密码 "radio123"
   - 或直接点击"游客模式访问"
4. 登录后会自动跳转到主应用页面

功能特点：
- 语音识别：点击"开始聆听"按钮说话控制
- 文字输入：在文本框中输入指令
- 快捷按钮：一键播放常用内容
- 智能推荐：根据您的需求推荐音频内容
- 用户认证：记住您的使用偏好
- 音频可视化：播放时显示动态效果

技术说明：
- 纯HTML/CSS/JS实现，无需服务器
- 语音识别使用浏览器原生API（需Chrome浏览器）
- 用户数据存储在本地浏览器中
- 支持音乐、新闻、广播剧等多种音频类型

注意事项：
- 首次使用语音功能需允许浏览器麦克风权限
- 推荐使用Chrome浏览器获得最佳体验
- 应用数据仅保存在本地，不会上传到任何服务器
